##Installation
前提としてnode.jsが必要です  
前提のパッケージとしてsocket.io,ntwitter,expressを要求します  
``` shell
$ npm　install socket.io ntwitter express
$ git clone https://github.com/alphaKAI/HarmonyQuartet.git
```
  
  
##Setting
app.jsの中にコンシューマーキーとアクセストークンを設定してください
